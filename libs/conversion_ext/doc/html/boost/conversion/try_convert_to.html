Function template try_convert_to

boost::conversion::try_convert_to

Synopsis

// In header: <boost/conversion/try_convert_to.hpp>


template<typename Target, typename Source> 
  enable_if_c< is_extrinsically_try_convertible_tagged< Source, Target >::value, optional< Target > >::type 
  try_convert_to(Source const & from);

Description

Effects: Converts the from parameter to an instance of the Target type, using the tyr_converter customization point.

Exception Safety: no-throw guarantee

Returns: A optional<Target> uninitialized when conversion fails.

Example: 

 optional<Target> t;
 Source s;
 res=boost::conversion::try_convert_to<Target>(s);
