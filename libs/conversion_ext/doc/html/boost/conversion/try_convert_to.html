Function template try_convert_to

boost::conversion::try_convert_to

Synopsis

// In header: <boost/conversion/try_convert_to.hpp>


template<typename Target, typename Source> 
  optional< Target > try_convert_to(Source const & from);

Description

Effects: Converts the from parameter to an instance of the Target type, using by default the conversion operator or copy constructor.

Exception Safety: no-throw guarantee

Returns: A optional<Target> uninitialized when conversion fails.
