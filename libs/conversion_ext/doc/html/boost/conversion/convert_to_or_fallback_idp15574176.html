Function template convert_to_or_fallback

boost::conversion::convert_to_or_fallback — Extrinsic conversion function with fallback. Converts the from parameter to a Target type. If the conversion fails the fallback value is used to construct a Target instance.

Synopsis

// In header: <boost/conversion/convert_to_or_fallback.hpp>


template<typename Target, typename Source, typename Fallback> 
  enable_if_c< !is_same< Target, Fallback >::value, Target >::type 
  convert_to_or_fallback(Source const & from, Fallback const & fallback);

Description

Returns: the converted value if the conversion succeeds or the fallback.

Throws: Whatever the conversion from Fallback to Target can throws when the conversion fails.

Remark(s): This function doesn't participate in function overload when the Fallback is the same as the Target.

Example: 

 std::string s="not an int";
 int t=boost::conversion::convert_to_or_fallback<int>(s,-1);
