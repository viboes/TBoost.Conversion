Function template try_assign_to

boost::conversion::try_assign_to — try to assign a target from a source

Synopsis

// In header: <boost/conversion/try_assign_to.hpp>


template<typename Target, typename Source> 
  bool try_assign_to(Target & to, const Source & from);

Description

Effects: Converts the from parameter to the to parameter, using the try_assigner customization point.

Exception Safety: no-throw guarantee

Returns: true if the conversion succeeds.

Example: 

 Target t;
 Source s;
 bool b = boost::conversion::try_assign_to(t,s);
