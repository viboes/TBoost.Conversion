boost::conversion::overload_workaround::convert_to_or_fallback

// In header: <boost/conversion/convert_to_or_fallback.hpp>

  // public static functions
  static To apply(const From &, Fallback const &);
};

Description

struct convert_to_or_fallback used when overloading can not be applied. This struct can be specialized by the user.

convert_to_or_fallback public static functions

  1. static To apply(const From & val, Fallback const & fallback);

    Effects: Converts the from parameter to an instance of the To type, using by default the conversion operator or copy constructor.

    Returns: the converted value if the conversion suceeds or the fallback.
