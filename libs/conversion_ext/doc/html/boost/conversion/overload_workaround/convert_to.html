Struct template convert_to

boost::conversion::overload_workaround::convert_to — struct used when overloading of convert_to function can not be applied.

Synopsis

// In header: <boost/conversion/convert_to.hpp>

template<typename To, typename From> 
struct convert_to {

  // public static functions
  static To apply(const From &);
};

Description

convert_to public static functions

  1. static To apply(const From & val);

    Effects: Converts the from parameter to an instance of the To type, using by default the conversion operator or copy constructor.

    Throws: Whatever the underlying conversion To operator of the From class or the copy constructor of the To class throws.
