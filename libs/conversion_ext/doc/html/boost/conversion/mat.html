Function template mat

boost::conversion::mat — makes an assignable to Target which accepts assignment from any type that is extrinsic assignable to Target.

Synopsis

// In header: <boost/conversion/assignable_to.hpp>


template<typename Target> assignable_to< Target > mat(Target & r);

Description

The result is able to transform conversion by convert_to calls and assignments by assign_to calls.

Exception Safety: no-throw guarantee.
