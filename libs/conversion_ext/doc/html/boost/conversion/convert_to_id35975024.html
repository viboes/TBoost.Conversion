Function template convert_to

boost::conversion::convert_to — Lazily convert to a type from an arbitrary argument.

Synopsis

// In header: <boost/conversion/fp/convert_to.hpp>


template<typename T, typename U> unspecified convert_to(U const & u);

Description

This overload of is taken in account when the parameter Source is a place_holder.

Returns: A unary functor that will call to the convert_to function on its parameter.

Throws: Whatever the underlying conversion To operator of the From class or the copy constructor of the To class throws.

Example: 

 boost::conversion::convert_to<int>(_1)(v);

Creates a functor that when applied to the parameter v, converts it to an int.
