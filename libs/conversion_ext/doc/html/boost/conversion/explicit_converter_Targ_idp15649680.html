Struct template explicit_converter<Target, Source, requires(ExtrinsicConvertible< Source, Target >)>

boost::conversion::explicit_converter<Target, Source, requires(ExtrinsicConvertible< Source, Target >)>

Synopsis

// In header: <boost/conversion/explicit_convert_to.hpp>

template<typename Target,   // target type of the conversion.
         typename Source  // source type of the conversion.
         > 
struct explicit_converter<Target, Source, requires(ExtrinsicConvertible< Source, Target >)> {

  // public member functions
  Target operator()(const Source &);
};

Description

Specialization for explicit_converter when is_explicitly_convertible<Source,Target>.

Requires: is_extrinsic_convertible<Source,Target>

explicit_converter public member functions

  1. Target operator()(const Source & val);

    Effects: Converts the from parameter to an instance of the Target type, using the conversion operator or copy constructor.

    Throws: Whatever the underlying conversion Target operator of the Source class throws.
