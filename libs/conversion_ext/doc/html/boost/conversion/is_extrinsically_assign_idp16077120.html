Struct template is_extrinsically_assignable

boost::conversion::is_extrinsically_assignable

Synopsis

// In header: <boost/conversion/is_extrinsically_assignable.hpp>

template<typename Target, typename Source> 
struct is_extrinsically_assignable {
};

Description

States if Target is extrinsically assignable from Source.

Condition: true_type if and only if the return expression in the following code would be well-formed: 

   assign_to(declval<Target>(), declval<Source>()); }

Requires: Target and Source must be complete types, (possibly cv-qualified) void, or arrays of unknown bound.
