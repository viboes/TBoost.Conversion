Function template implicit_convert_to

boost::conversion::implicit_convert_to — Extrinsic conversion function. Converts the from parameter to an instance of the Target type. This function can be seen as an emulation of free function overload of the conversion operator.

Synopsis

// In header: <boost/conversion/implicit_convert_to.hpp>


template<typename Target, typename Source> 
  Target implicit_convert_to(Source const & from);

Description

Parameters:

- source: source of the conversion.

Returns: The result of converter customization point.

Throws: Whatever the converter call operator throws.
