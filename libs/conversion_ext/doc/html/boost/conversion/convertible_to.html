Description

convertible_to public construct/copy/destruct

  1. template<typename Source> convertible_to(Source const & source);

    Constructor from a extrinsic implicitly convertible to Target.

    Effects: Store the extrinsic conversion from source to @ Target.

    Throws: Whatever extrinsic implicit conversion from source to Target could throw.

    Remark(s): This constructor doesn't participates on overload resolution if Source is not extrinsic convertible to Target.
