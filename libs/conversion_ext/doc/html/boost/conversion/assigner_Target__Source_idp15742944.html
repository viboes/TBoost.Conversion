boost::conversion::assigner<Target, Source, requires(CopyAssignable< Target > &&ExtrinsicallyExplicitConvertible< Source, Target > &&!Assignable< Target, Source >)>

Synopsis

// In header: <boost/conversion/assign_to.hpp>

template<typename Target,   // target type of the conversion.
         typename Source  // source type of the conversion.
         > 
struct assigner<Target, Source, requires(CopyAssignable< Target > &&ExtrinsicallyExplicitConvertible< Source, Target > &&!Assignable< Target, Source >)> {

  // public member functions
  Target & operator()(Target &, const Source &);
};

Description

Specialization when Target is not assignable from Source, but Target is copy constructible and Source is extrinsic convertible to Target.

Requires: Target must be CopyAssinable and Source must be extrinsic convertible to Target.

assigner public member functions

  1. Target & operator()(Target & to, const Source & from);

    Effects: Converts the from parameter to the to parameter, using by default the assignment operator.

    Throws: Whatever the underlying assignment operator of the Target class throws.
