Struct template assigner<Target, Source, requires(Assignable< Target, Source >)>

boost::conversion::assigner<Target, Source, requires(Assignable< Target, Source >)>

Synopsis

// In header: <boost/conversion/assign_to.hpp>

template<typename Target,   // target type of the conversion.
         typename Source  // source type of the conversion.
         > 
struct assigner<Target, Source, requires(Assignable< Target, Source >)> {

  // public member functions
  Target & operator()(Target &, const Source &);
};

Description

Specialization when Target is assignable from Source.

Requires: Target must be Assinable from Source.

assigner public member functions

  1. Target & operator()(Target & to, const Source & from);

    Effects: Assigns the from parameter to the to parameter.

    Throws: Whatever the underlying assignment operator of the Target class throws.
