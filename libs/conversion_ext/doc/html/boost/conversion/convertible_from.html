Class template convertible_from

boost::conversion::convertible_from — wrapper providing implicit conversion operation to any type extrinsicaly implicit convertible from Source.

Synopsis

// In header: <boost/conversion/convertible_from.hpp>

template<typename Source, 
         typename Enable = requires(            CopyConstructible<Source>        )> 
class convertible_from {
public:
  // construct/copy/destruct
  convertible_from(Source);

  // public member functions
  template<typename Target, 
           typename boost::enable_if< is_extrinsically_convertible< Source, Target >, int >::type = 0> 
    operator Target() const;
};

Description

This wrapper is used indirectly through the function to pass a Source parameter to a function waiting a parameter Target extrinsicaly convertible from it.

Requires Source must be CopyConstructible

convertible_from public construct/copy/destruct

  1. convertible_from(Source source);
    constructor from a Source.

    Throws: Whatever the Source copy constructor could throw.

convertible_from public member functions

  1. template<typename Target, 
         typename boost::enable_if< is_extrinsically_convertible< Source, Target >, int >::type = 0> 
  operator Target() const;
    Implicit conversion to a Target extrinsicaly convertible from Source.

    Effects: Forwards the conversion from the reference using conver_to.

    Returns: the conversion using conver_to

    Throws: Whatever extrinsic conversion from Source to Target could throw.

    Remark(s): On compilers that supports C++0x default arguments for function template parameters, this conversion operator doesn't participates on overload resolution if Source is not extrinsic convertible to Target.
