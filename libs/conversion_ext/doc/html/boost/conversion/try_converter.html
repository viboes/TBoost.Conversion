boost::conversion::try_converter

Synopsis

// In header: <boost/conversion/try_convert_to.hpp>

template<typename Target,   // target type of the conversion.
         typename Source,   // source type of the conversion.
         typename Enable = void  // A dummy template parameter that can be
                                 // used for SFINAE.
         > 
struct try_converter {

  // public member functions
  optional< Target > operator()(const Source &);
};

Description

Customization point for try_convert_to.

This struct can be specialized by the user.

try_converter public member functions

  1. optional< Target > operator()(const Source & val);

    Requires: Target must be CopyConstructible and ::boost::conversion::convert_to<Target>(from) must be well formed.

    Effects: Converts the from parameter to an instance of the Target type, using by default the conversion operator or copy constructor.

    Exception Safety: no-throw guarantee

    Returns: A optional<Ratget> uninitialized when conversion fails.
