Struct template is_copy_assignable

boost::is_copy_assignable

Synopsis

// In header: <boost/conversion/type_traits/is_copy_assignable.hpp>

template<typename T> 
struct is_copy_assignable : public boost::is_assignable< T &, T const & > {
};

Description

States if T is copy assignable.

Condition: is_assignable<T&, T const&>::value is true.

Requires: T must be a complete type, (possibly cv-qualified) void, or an array of unknown bound.
