Function template convert_to

boost::convert_to

Synopsis

// In header: <boost/conversion/convert_to.hpp>


template<typename Target, typename Source> 
  Target convert_to(Source const & from, 
                    boost::dummy::base_tag< Target > const & p = boost::dummy::base_tag< Target >());

Description

Effects: Converts the from parameter to an instance of the To type, using by default the conversion operator or copy constructor.

Throws: Whatever the underlying conversion To operator of the From class or the copy constructor of the To class throws. This function can be partialy specialized on compilers supporting it. A trick is used to partialy specialize on the return type by adding a dummy parameter.
