Function template assign_to

boost::conversion_2::assign_to — Default assign_to overload, used when ADL fails.

Synopsis

// In header: <boost/conversion/assign_to.hpp>


template<typename To, typename From> 
  To & assign_to(To & to, const From & from, dummy::type_tag< To > const &);

Description

Effects: Converts the from parameter to the to parameter, using by default the assigment operator.

Throws: Whatever the underlying the assignment operator of the To class throws.. Forwards the call to the overload workarround, which can yet be specialized by the user for standard C++ types.
