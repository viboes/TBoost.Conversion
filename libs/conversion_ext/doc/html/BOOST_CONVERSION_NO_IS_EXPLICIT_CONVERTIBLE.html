Macro BOOST_CONVERSION_NO_IS_EXPLICIT_CONVERTIBLE

BOOST_CONVERSION_NO_IS_EXPLICIT_CONVERTIBLE

Synopsis

// In header: <boost/conversion/type_traits/is_explicitly_convertible.hpp>

BOOST_CONVERSION_NO_IS_EXPLICIT_CONVERTIBLE

Description

Macro stating if the compiler doesn't support the features needed to define the is_explicitly_convertible type trait using the static_cast version. In this case the trait is equivalent to is_constructible<Target,Source>.
